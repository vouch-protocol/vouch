# Changelog

## 0.1.0 - 2026-05-23

Initial release.

- Command: **Vouch: Insert local-first quickstart at cursor** (Python, TypeScript, Go).
- Command: **Vouch: Generate a did:web identity** (opens terminal, runs `vouch init`).
- Command: **Vouch: Ask the assistant** (opens `vouch-protocol.com/ask` in browser).
- IntelliSense snippets: `vouch-quickstart`, `vouch-hybrid-pq`, `vouch-middleware`.
- Status-bar shortcut: a single click filters the command palette to Vouch commands.
- Two settings: `vouch.statusBar.enabled`, `vouch.assistant.url`.
- Zero telemetry, zero auto-scan, zero network calls unless a command is invoked.
