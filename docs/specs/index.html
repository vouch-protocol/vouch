Specifications & Reports

Vouch Protocol is on a W3C standards track. This page lists the active draft specifications, the published Community Group Report, the IETF reference documentation for the legacy v0.x JWS profile, and supporting briefs aimed at standards body voting members and prospective sponsors.

Primary specification

W3C Community Group Report

Active draft Updated 26 April 2026

The current v1.6 normative specification. Defines the Vouch Credential format (W3C VC + Data Integrity), Multikey verification methods, delegation chains with explicit resource binding, the Identity Sidecar pattern, the Heartbeat Protocol, and the optional hybrid post-quantum profile.

Companion specifications

Hybrid Post-Quantum Implementation Guide

Active `hybrid-eddsa-mldsa44-jcs-2026`

Implementer guide for the optional hybrid Ed25519 + ML-DSA-44 cryptosuite. Three verifier modes, dependency setup for Python / TypeScript / Go, performance and size considerations, and the cross-implementation test vector.
