# TypeScript SDK Examples

This directory contains examples demonstrating all features of the Vouch TypeScript SDK.

## Installation

```bash
npm install @vouch-protocol/sdk
```

## Examples

### Basic Usage
- `basic-connect.ts` - Connect to daemon and check status
- `basic-sign.ts` - Sign text content
- `sign-with-metadata.ts` - Sign with custom metadata

### File Operations
- `sign-blob.ts` - Sign binary data (Blob/File)
- `sign-form-upload.ts` - Sign file from form input

### Error Handling
- `error-handling.ts` - Graceful error handling patterns

### Framework Integrations
- `nextjs-integration.ts` - Use with Next.js API routes
- `express-integration.ts` - Use with Express.js

### Browser Usage
- `browser-basic.html` - Basic browser usage (no build step)
- `react-component.tsx` - React component example
